# 💰  **Bolso em dIA**

Assistente financeiro inteligente desenvolvido com IA generativa, focado em educação financeira e simulações práticas de investimento.

  

---

##  Sobre o projeto

O **Bolso em dIA** é uma aplicação interativa que utiliza inteligência artificial para responder dúvidas financeiras, realizar simulações de investimento e orientar o usuário de forma simples e clara.

O projeto foi desenvolvido como parte de uma iniciativa educacional em parceria com a **DIO + Bradesco**, com foco em aplicar conceitos de:

* IA generativa
* Processamento de linguagem natural
* Python
* Visualização de dados
* Experiência do usuário (UX)

---

##  Funcionalidades

✔ Assistente financeiro com IA local
✔ Respostas contextualizadas
✔ Simulação de juros compostos
✔ Interpretação de linguagem natural
✔ Gráfico de evolução do investimento 📈
✔ Filtro inteligente (responde apenas sobre finanças)

---

##  Exemplo de uso

Entrada do usuário:

> investir 200 por 12 meses a 1%

Saída:

💰 Simulação de investimento:

📌 Valor mensal: R$200
⏱️ Tempo: 12 meses
📈 Taxa: 1% ao mês

💵 Valor final: R$2536.50

* gráfico de crescimento do investimento 📈

##  Tecnologias utilizadas

* Python

##  Como executar o projeto

### 1. Clone o repositório

```
git clone https://github.com/seu-usuario/bolso-em-dia.git
```

### 2. Acesse a pasta

```
cd bolso-em-dia
```

### 3. Crie o ambiente virtual

```
python -m venv venv
```

### 4. Ative o ambiente

Windows:

```
.\venv\Scripts\Activate.ps1
```

### 5. Instale as dependências

```
pip install streamlit requests
```

### 6. Execute o Ollama

```
ollama run phi
```

### 7. Rode a aplicação

```
streamlit run app.py
```
